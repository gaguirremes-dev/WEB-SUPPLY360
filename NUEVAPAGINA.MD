FORMA DE TRABAJAR

Antes de escribir código:

1. Analiza toda la estructura existente.
2. Identifica componentes reutilizables.
3. Propón una nueva arquitectura de carpetas.
4. Diseña un sistema de diseño (Design System) con tokens para colores, tipografía, espaciado, radios, sombras y animaciones.
5. Define una biblioteca de componentes reutilizables (botones, tarjetas, secciones, badges, CTAs, grids, navegación, footer, etc.).
6. Presenta un wireframe textual de todas las secciones.
7. Después implementa el HTML semántico, CSS modular y JavaScript organizado por módulos ES6.
8. Documenta el código y explica las decisiones de diseño.

No reutilices la apariencia actual. Conserva únicamente el contenido útil y los recursos multimedia. El resultado debe sentirse como un sitio completamente nuevo, moderno y preparado para escalar.
